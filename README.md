# stocks price prediction with Tensorfow-2 and Keras

## Overview
This project uses Deep Learning to predict the opening price of Microsoft's stocks.

## Requirements
-   Python 3.5+
-   alpha_vantage
-   pandas
-   numpy
-   sklearn
-   keras
-   tensorflow2
-   matplotlib

-   Run eval.py to evaluate the model
